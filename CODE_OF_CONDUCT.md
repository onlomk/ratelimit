# Code of Conduct

This project follows a simple code of conduct: be respectful, constructive, and professional.

## Expected behavior

- Use welcoming and inclusive language.
- Respect different viewpoints and experience levels.
- Provide constructive feedback.
- Focus on what is best for the project and its users.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Publishing private information without permission.
- Spam or intentionally disruptive behavior.

Maintainers may remove comments, issues, or pull requests that violate these expectations.
